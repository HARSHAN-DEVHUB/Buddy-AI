# Buddy AI

Welcome to Buddy AI, your intelligent assistant designed to interact with GitHub data on your behalf.

## Features
- Create and update files
- Manage branches
- Merge pull requests

## Installation
To install Buddy AI, clone the repository and run the installation command:

```bash
git clone https://github.com/HARSHAN-DEVHUB/Buddy-AI.git
cd Buddy-AI
```
